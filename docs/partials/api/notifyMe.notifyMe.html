notifyMe
service in module notifyMe

Description

This service contains apis to create beautiful notifications.

Methods

  • clear(notifObj)

    Dismisses or clears notifications. If notification object is passed, it dismisses that notification else it will clear all notifications.

    Parameters
    ParamTypeDetails
    notifObjnotificationObject

    (Optional) Notification object which needs to be cleared.

    Example

    notifyMe.clear([notifObj]);

  • ofError(msg, title)

    Method to create ERROR notification.

    Parameters
